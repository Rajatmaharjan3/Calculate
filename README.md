# Calculate
This is a basic calculator application built using Python. It provides a clean and user-friendly interface to perform common arithmetic operations such as addition, subtraction, multiplication, and division.

Features
Perform basic operations: +, -, ×, ÷
Percentage calculation (%)
Square root function (√)
Toggle positive/negative (+/-)
Clear all input (AC)
Decimal number support
Simple and intuitive UI design

Technologies Used
Python
GUI library (e.g., Tkinter / PyQt / Kivy — update based on what you used)

Purpose
This project is designed for beginners to understand:
GUI development in Python
Event handling (button clicks)
Basic logic building for arithmetic operations
